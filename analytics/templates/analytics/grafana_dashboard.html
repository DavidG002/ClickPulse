ClickPulse: ClickHouse Analytics Demo

Welcome to ClickPulse, a lightweight demo showcasing the power of ClickHouse—an open-source, column-oriented database—for real-time analytics. This app integrates ClickHouse with Grafana to visualize sales data from a CSV file, featuring a real-time simulator that updates the dashboard live. Built with simplicity and speed in mind, it demonstrates how ClickHouse can handle fast, scalable analytics for modern applications. Check out the source code and details on GitHub.
