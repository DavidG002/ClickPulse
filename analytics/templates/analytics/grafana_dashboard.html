ClickPulse: ClickHouse Analytics Demo

Welcome to ClickPulse, a dynamic demo showcasing ClickHouse—an open-source, column-oriented database—for real-time analytics. This app ingests large, complex CSV files—spanning sales, finance, e-commerce, statistics, logistics, IoT, and raw operational data—into ClickHouse and visualizes it with Grafana. The dashboard below uses real e-commerce data from the UCI “Online Retail” dataset, featuring over 500,000 transactions from a UK retailer (2010-2011), highlighting gift sales trends. Built to demonstrate simplicity, speed, and scalability, it shows how ClickHouse transforms data into actionable insights. Dive into the source code and details on GitHub.
